# PLATO-overwrite-emulator

Quick and dirty emulator of PLATO's overwrite capability

## Capabilities

- [x] CSS overlapping
- [ ] Use correct font
- [ ] Populate according to text input
- [ ] Font selector
- [ ] Update colours to match PLATO
