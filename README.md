# PLATO-overwrite-emulator

Quick and dirty emulator of PLATO's overwrite capability

## Capabilities

- [x] CSS overlapping
- [x] Use correct font
- [ ] Populate according to text input
- [ ] Font selector
- [x] Update colours to match PLATO
- [ ] Setup GitHub Pages, subdomain, etc
